# <a name="microsoft-store-marketing-guidelines-for-apps"></a>市场营销应用准则的 Microsoft Store

了解如何升级你的应用和 Microsoft Store 中的内容。 这些指南涉及如何使用可用资源以及有关在打印、电视、社交媒体和数字广告中推广应用的建议。

## <a name="store-badges"></a>应用商店锁屏提醒

我们创建了特殊促销徽章来帮助您更多客户到 Microsoft Store 中的应用程序的列表。 这些锁屏提醒在 44 种语言中可用。 请记住，使用这些图像时需要遵循一些要求；这些要求以及使用示例和指南将以 PDF 文件形式提供。

[![下载按钮](images/downloadbutton.png)](https://go.microsoft.com/fwlink/p/?LinkId=529769) **Microsoft Store 徽章指南 （PDF、 英语）**


## <a name="badge-generator-and-images"></a>锁屏提醒生成器和图像

你可以使用[锁屏提醒生成器](https://go.microsoft.com/fwlink/p/?LinkID=534236)生成显示（使用所选语言）应用商店锁屏提醒的 HTML 和直接指向应用的应用商店列表的链接。

你也可以从以下链接中下载完整的锁屏提醒图像集（格式为 PNG 和 PDF）。

[![下载按钮](images/downloadbutton.png)](https://go.microsoft.com/fwlink/p/?LinkId=529771)**所有标记映像**


## <a name="device-images"></a>设备图像

若要推广应用，你可能会想展示其在 Windows 设备上运行时的外观。 我们提供了各种设备底盘图像供你选择，包括手机、平板电脑、笔记本电脑和 PC。 可以在找到最佳实践和如何使用这些映像的技术规格[Microsoft Store 市场营销准则](https://go.microsoft.com/fwlink/p/?LinkId=529769)。

[![下载按钮](images/downloadbutton.png)](https://go.microsoft.com/fwlink/p/?LinkId=533057) **Windows 设备艺术**

## <a name="license-to-microsoft-marks"></a>Microsoft 商标授权

*Microsoft 标记*意味着*Microsoft 锁屏提醒*所述[徽章生成器](https://go.microsoft.com/fwlink/p/?LinkID=534236)页。 若要使用这些锁屏提醒，必须满足以下条件：

-   在 Microsoft Store 中，您的应用程序或其他可用的内容或属于[Microsoft 关联计划](https://go.microsoft.com/fwlink/p/?LinkId=624463)。

-   如果注册为应用开发人员在合作伙伴中心，符合["到 Microsoft 标记的许可证"](https://docs.microsoft.com/legal/windows/agreements/app-developer-agreement#license_to_mark)应用开发人员协议的一部分。

-   如果您未注册为合作伙伴中心中的应用开发人员，Microsoft 向你授予 Microsoft Store 徽章指南中所述的仅使用徽章的全球范围内、 非排他性、 不可转让、 免版税的许可证。 Microsoft 可能会更改这些指南，如果要更改，Microsoft 会通过合理的方式将你重定向到任何发布这些规范的新 URL。 Microsoft 保留此处未明确授予的所有权利。

-   请按照中所述的徽标使用规范[Microsoft Store 徽章指南](https://go.microsoft.com/fwlink/p/?LinkId=529769)。
