# <a name="create-customer-groups"></a>创建客户组

可通过创建包含目标客户的*客户组*，使用 Windows 开发人员中心仪表板面向应用的客户群子集。 这些组对于促销、测试或其他用途可能很有用。

若要查看和创建客户组，请在左侧导航菜单中展开**参与**，然后选择仪表板顶部附近的**客户组**。

当前支持两种客户组：

- **客户类别。** 这些是动态创建的 Windows10 客户组，满足所选的人口统计或收益条件。 类别通常与[推送通知](send-push-notifications-to-your-apps-customers.md)或[定向优惠](use-targeted-offers-to-maximize-engagement-and-conversions.md)结合使用。 有关详细信息，请参阅[创建客户类别](create-customer-segments.md)。
- **已知用户组。** 这些是特定客户构成的组，根据与其 Microsoft 帐户关联的电子邮件地址创建而来。 已知用户组最常与[软件包外部测试版](package-flights.md)结合使用，以向该组中的客户提供特定软件包。 有关详细信息，请参阅[创建已知用户组](create-known-user-groups.md)。
