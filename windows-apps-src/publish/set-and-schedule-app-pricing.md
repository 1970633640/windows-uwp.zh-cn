# <a name="set-and-schedule-app-pricing"></a>设置和计划应用定价

可通过[定价和可用性](set-app-pricing-and-availability.md)页面的**定价**部分为应用选择一个基价。 还可[计划价格更改](#schedule-price-changes)，以指示应用价格应更改的日期和时间。 此外，还可选择[针对特定市场自定义这些更改](#customize-pricing-for-specific-markets)。 

> [!NOTE]
> 虽然本主题涉及应用，但加载项提交的价格选择也使用同样的过程。

## <a name="base-price"></a>基价

选择应用的**基价**后，该价格将在所有出售应用的市场使用，除非为特定市场指定了自定义价格。

可将**基价**设置为**免费**，或可选择一个价格段，该价格段将确定所选择要发布应用的所有国家/地区中的销售价格。 价格段从 0.99 美元开始，并设有按一定增量递增的其他价格（1.09 美元、1.19 美元等）。 价格越高，增量越大。 

> [!NOTE]
> 这些价格段也适用于加载项。 

应用商店为每个价格段都提供了 60 多种不同货币对应值。 我们使用这些值来帮助你在世界范围内以适当价格销售应用。 可选择以任何货币表示的价格，我们会自动在不同的市场使用这些价格的对应值。

在**定价**部分中，单击**查看表**，可查看以每种货币表示的相应价格。 其中还会显示与每个价格段关联的 ID 号，如果使用[Windows 应用商店提交 API](../monetize/manage-app-submissions.md#price-tiers)输入价格，则需要使用这个号码。 可单击**下载**，以 .csv 文件格式下载价格段表的一个副本。

请记住，你选择的价格段可能包含你的客户必须支付的销售税或增值税。 若要了解有关选定市场中应用的税收含义的详细信息，请参阅[付费应用的税收详细信息](tax-details-for-paid-apps.md)。 还应查看[特定市场的价格考虑因素](define-pricing-and-market-selection.md#price-considerations-for-specific-markets)。

## <a name="schedule-price-changes"></a>计划价格更改

如果希望在特定日期和时间更改应用基价，可选择计划一个或多个价格更改。 

> [!IMPORTANT]
> 仅向使用 Windows 10 的客户显示价格更改。 如果你的应用支持较早版本的操作系统，则价格更改不适用。 
>
> - 对于使用 Windows 8 的客户，应用将始终按**基价**（非任何特定于市场的价格）提供应用，即使计划了其他价格更改也是如此。 
> - 对于使用 Windows 8.1 和 Windows Phone 8.1 及更早版本的客户，应用将始终按客户所在市场的初始价格提供，即使在该市场计划了其他价格更改时也是如此。
> 
> 计划价格更改时请记住这一点。 例如，如果最初以较低价格发布应用，然后计划了涨价日期，则使用较早版本的操作系统购买应用的客户将按较低（初始）价格购入应用。

单击**计划价格更改**可查看价格更改选项。 选择要使用的价格段，然后选择日期、时间和时区。

可单击**再次计划价格更改**，根据所需计划后续更改。

> [!NOTE]
> 计划的价格更改与[销售定价](put-apps-and-add-ons-on-sale.md)的运作方式不同。 将应用进行促销时，查看应用商店一览的客户将看到价格已降低，并且他们能够在你选择的时间段期间以较低的价格购买它。 促销期结束后，价格将复原为基价。
>
> 通过计划价格变更，可将价格调高或调低。 更改将在指定日期生效，但应用商店中不会显示该应用正在促销；该应用只是会具有一个新的基价。 

## <a name="customize-pricing-for-specific-markets"></a>针对特定市场自定义定价

默认情况下，上述所选选项将应用于所有提供应用的市场。 若要针对特定市场自定义价格，请选择**针对特定市场自定义**。 随即将出现**市场选择**弹出窗口，其中列出了所有你选择要提供应用的市场。 如果在“市场”部分中排除了这些市场，则此处将不会显示这些市场。 

> [!IMPORTANT]
> 使用 Windows 8 的客户所看到的应用价格将始终为应用的**基价**，即使为其所在市场选择一个不同的价格也是如此。

若要为某个市场添加自定义定价，请选中该市场，然后单击**保存**。 然后仍会看到与之前一样的**基价**和**计划价格更改**选项，如上所述，但所做选择将特定于该市场。

若要为多个市场添加自定义定价，需创建*市场组*。 为此，请选择要包括的市场，然后为该组输入名称。 （此名称仅供你自己参考，对客户不可见。）完成后，单击**保存**。 然后将出现与之前一样的**基价**和**计划价格更改**选项，如上所述，但所做选择将特定于该市场组。

> [!NOTE]
> 一个市场不能同时属于“定价”部分中所使用的多个市场组。

若要为其他市场或市场组添加自定义定价，请再次选择**针对特定市场自定义**并重复上述步骤。 若要更改包括在市场组中的市场，请选择其名称。 若要为市场组（或个别市场）删除定价，请单击**删除**。
