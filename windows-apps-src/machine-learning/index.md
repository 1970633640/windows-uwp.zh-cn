# <a name="windows-ml"></a>Windows ML

Windows 机器学习 (ML) 让你能够在你的应用中使用经过训练的机器学习模型。 该平台在 Windows 10 设备本地评估经过训练的模型，通过利用设备的 CPU 或 GPU 提供硬件加速性能，并针对经典 ML 算法和深度学习计算评估。
