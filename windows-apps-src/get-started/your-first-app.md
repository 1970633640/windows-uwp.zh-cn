# 创建你的第一个应用

欢迎访问：这些教程将帮助你使用你选择的语言创建你的第一个 UWP 应用。 在此处，你将了解如何：

-   在 Microsoft Visual Studio 中创建 UWP 应用项目。
-   将 UI 元素和代码添加到项目。
-   在本地计算机和手机模拟器上生成和调试应用。

若要开始操作，请选择你想使用的语言。
