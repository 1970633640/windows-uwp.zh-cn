# 盈利、客户参与度和应用商店服务




Windows SDK、Microsoft Store Services SDK 和 Windows 应用商店提供的功能可使你从应用中获取更多收益，并通过吸引用户来赢得客户。 本部分的主题介绍了如何将这些功能生成到你的应用中。

有关 Windows 应用商店收费情况以及如何获取应用销售所得的详细信息，请参阅[获得收入](https://msdn.microsoft.com/library/windows/apps/mt148536)。
