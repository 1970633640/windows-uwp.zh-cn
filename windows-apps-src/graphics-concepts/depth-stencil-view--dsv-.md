# <a name="depth-stencil-view-dsv"></a>深度模具视图 (DSV)


深度模板视图提供用于保存深度和模板信息的格式和缓冲区。 深度缓冲区用于剔除由较近的对象对其进行遮挡时，对查看者不可见的像素的绘制。 模具缓冲区可用于剔除定义的形状以外的所有绘制。

除了定义渲染区域外，模具缓冲区还有许多更高级的用法。 可以操作模具缓冲区值以提供淡化、剪影、贴纸、溶解、描绘轮廓、阴影卷等效果。
