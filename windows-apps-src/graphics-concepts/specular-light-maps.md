# <a name="specular-light-maps"></a>高光映射


通过光源照亮时，采用高度发光材料的闪光对象会接收反射高光。 有时，相较于使用照明模块产生的反射高光，你可通过将高光映射应用至基元获取更精确的高光。

将高光映射添加至基元纹理，然后进行调制（将结果乘以）RGB 光映射，以进行高光映射。
