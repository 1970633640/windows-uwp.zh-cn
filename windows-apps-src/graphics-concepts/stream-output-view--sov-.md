# <a name="stream-output-view-sov"></a>流输出视图 (SOV)


流输出视图支持将顶点、分割和几何图形着色器附带的顶点信息流式传输回应用程序以供进一步使用。 例如，已经由这些着色器导致失真的对象可以写回到应用程序，从而为物理引擎或其他引擎提供更准确的输入。 但在实践中，流输出视图是图形管道中较少用到的功能。
