# <a name="stream-output-view-sov"></a>流输出视图 (SOV)


流输出视图可输出顶点、分割和几何图形着色器附带的顶点信息，使其回到应用程序以做进一步使用。 例如，已经因着色器导致失真的对象可以写回应用程序，从而为物理或其他引擎提供更准确的输入。 但在实践中，流输出视图是图形管道中较少用到的功能。
