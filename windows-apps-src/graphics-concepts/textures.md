# <a name="textures"></a>纹理


纹理是在计算机生成的 3D 图像中创建逼真效果的强大工具。 Direct3D 支持广泛的纹理功能集，使开发人员能够轻松访问高级纹理技术。

为提升性能，可以考虑使用动态纹理。 动态纹理可以锁定、写入以及解锁每个帧。
