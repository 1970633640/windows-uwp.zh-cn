# <a name="diffuse-light-maps"></a>漫射光映射


亚光表面具有漫反射光。 漫反射光的亮度取决于与光源的距离以及表面法线与光源方向矢量之间的角度。 纹理光照贴图可以模拟复杂的漫射照明。

光线计算模拟的漫射光线效果仅会产生常规效果。 你的应用程序可以使用纹理光线映射模拟更复杂的漫射光线。
