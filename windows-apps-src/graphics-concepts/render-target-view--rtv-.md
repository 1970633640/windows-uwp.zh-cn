# <a name="render-target-view-rtv"></a>呈现目标视图 (RTV)


呈现目标使场景能够呈现到临时中间缓冲区，而不是呈现到要呈现到屏幕的后台缓冲区。 利用此功能，可在图形管道中将可呈现的复杂场景用作反射纹理或用于其他用途，也可将其用于在呈现前向场景添加其他像素着色器效果。
