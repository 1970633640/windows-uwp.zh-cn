# <a name="depth-and-stencil-buffers"></a>深度和模板缓冲区


*深度缓冲区*存储深度信息，以控制渲染（而不是从视图中隐藏）哪些多边形区域。 *模板缓冲区*用于遮罩图像中的像素，以产生特殊效果，包括合成；贴纸；溶解、淡化和滑动；轮廓描绘和剪影，以及双面模板。
