# 应用设置和数据




本部分包含显示应用设置和将这些设置存储为应用数据的用户体验指南。

应用设置是通用 Windows 平台 (UWP) 应用的用户可自定义的部分。 例如，新闻阅读器应用可让用户指定显示的新闻源或屏幕上显示的列数。

应用数据是应用自身创建和管理的数据。 它包含运行时状态、应用设置、参考内容（如字典应用中的字典定义）以及其他设置。 应用数据以应用的存在为前提，并且仅对该应用有意义。
