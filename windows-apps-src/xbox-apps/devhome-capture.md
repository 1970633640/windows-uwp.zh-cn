# <a name="media-capture-page-dev-home"></a>媒体捕获页 （开发主页）
   
  
开发主页的**媒体捕获**页捕获控制台当前运行的标题的视频。   
 ![捕获页开发主页](images/devhome_capture.png)   
  
**媒体捕获**页包含以下窗格：   
 
   *  [视频捕获](#ID4EHB)  

 
<a id="ID4EHB"></a>

   

## <a name="video-capture"></a>视频捕获  
   
  
**视频捕获**窗格包含记录从开发人员控制台的视频流的工具。 您可以捕获从控制台的最后几秒的视频。
