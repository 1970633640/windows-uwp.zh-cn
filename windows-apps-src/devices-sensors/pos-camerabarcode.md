# <a name="camera-barcode-scanner"></a>相机条形码扫描仪
相机条形码扫描仪在 Windows 将连接到计算机的相机与软件解码器配对时动态创建。  每个相机 - 解码器对都是一个功能完全的条形码扫描仪。
