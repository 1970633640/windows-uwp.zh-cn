# <a name="3d-printing"></a>3D 打印


本部分介绍如何利用[Windows.Graphics.Printing3D 命名空间](https://docs.microsoft.com/uwp/api/windows.graphics.printing3d)将 3D 打印功能添加到通用 Windows 应用程序。  

有关与 Windows 10 3D 打印的详细信息，包括资源有关的硬件合作伙伴、 社区讨论论坛和 3D 打印功能的一般信息，请参阅[与 Windows 10 3D 打印](https://developer.microsoft.com/windows/hardware/3d-print/windows-3d-printing)在硬件上的站点开发人员中心。

| 主题 | 描述 |
|-------|-------------|
| [从应用进行 3D 打印](3d-print-from-app.md) | 了解如何将 3D 打印功能添加到你的 UWP 应用。 本主题介绍了如何在确保 3D 模型可打印并且格式正确后启动 3D 打印对话框。 |
| [生成 3MF 程序包](generate-3mf.md) | 描述三维制造格式文件类型和如何可以创建和使用 Windows.Graphics.Printing3D Api 操作的结构。 |
