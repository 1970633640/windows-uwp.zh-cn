# <a name="xbox-live-storage-platform---connected-storage-title-storage"></a>Xbox Live 存储平台 - 连接存储、标题存储

借助 Xbox Live，发布者能够在云中存储全局游戏数据和特定于玩家的数据。

## <a name="in-this-section"></a>本部分内容

[连接存储](connected-storage/connected-storage-technical-overview.md)  
采用连接存储 API 存储的数据会自动漫游，可供用户跨电脑和多个 Xbox One 主机使用，并且还能脱机使用。

[Xbox Live 标题存储](xbox-live-title-storage/xbox-live-title-storage.md)  
Xbox Live 标题存储服务提供了一种在云中存储标题游戏信息的方法。 所有平台上运行的游戏均可使用此服务。
