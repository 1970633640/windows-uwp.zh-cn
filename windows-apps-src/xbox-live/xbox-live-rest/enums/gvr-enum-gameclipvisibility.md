# <a name="gameclipvisibility-enumeration"></a>GameClipVisibility 枚举
详细介绍 GameClipVisibility 枚举。 
<a id="ID4ER"></a>

 
## <a name="gameclipvisibility"></a>GameClipVisibility
 
| <b>枚举器</b>| <b>描述</b>| 
| --- | --- | 
| 无| 未指定可见性。 这是在大多数情况下一个无效值。| 
| 默认| 游戏剪辑可见性由 Xbox LIVE 隐私设置。| 
| 所有者| 游戏剪辑才可用于创建原始 XUID。| 
| Title| 游戏剪辑功能仅适用于创建它的原始标题。| 
| Public| 可供任何人在 Xbox LIVE 游戏剪辑。|
