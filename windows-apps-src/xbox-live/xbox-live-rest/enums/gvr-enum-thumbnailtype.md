# <a name="thumbnailtype-enumeration"></a>ThumbnailType 枚举
详细介绍 ThumbnailType 枚举。 
<a id="ID4ER"></a>

 
## <a name="thumbnailtype"></a>ThumbnailType
 
| <b>枚举</b>| <b>说明</b>| 
| --- | --- | 
| 无| 未指定缩略图的类型。 这是在大多数情况下无效的值。| 
| 小| 指定较小缩略图。 小是 424 像素 208 像素的宽度。| 
| 大| 指定较大的缩略图。 大是 856 像素 424 像素的宽度。|
