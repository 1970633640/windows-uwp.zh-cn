# <a name="usersxuidxuidgroupsmonikerbroadcastingcount"></a>/ 用户/xuid ({xuid}) /groups/ {名字对象} / 广播/计数

## <a name="uri-parameters"></a>URI 参数
 
| 参数| 类型| 说明|

| 名字对象| 字符串| 定义的用户组的字符串。 目前仅接受名字对象以大写 P 是"People"。|
