# <a name="get-inventoryitemid"></a>获取 (/inventory/ {itemID})
提供特定的清单项的详细信息的一整套。 这些 Uri 的域是`inventory.xboxlive.com`。
 
  * [备注](#ID4EX)
  * [URI 参数](#ID4EAB)
  * [响应正文](#ID4ELB)
 
<a id="ID4EX"></a>

 
## <a name="remarks"></a>备注
 
没有策略检查，强制执行，或筛选会作为此调用的一部分。
  
<a id="ID4EAB"></a>

 
## <a name="uri-parameters"></a>URI 参数
 
| 参数| 类型| 说明| 
| --- | --- | --- | 
| itemID| 字符串| 每个用户的单一库存项目的唯一 ID|
