# <a name="achievement-title-history-uris"></a>游戏成就历史记录 URI
 
本部分提供了有关通用资源标识符 (URI) 地址和关联的超文本传输协议 (HTTP) 方法的详细信息，Xbox Live 服务的成就相关*标题历史记录*。 在所有平台上运行的游戏可以读取此历史信息。
 
这些 Uri 的域是 achievements.xboxlive.com。
