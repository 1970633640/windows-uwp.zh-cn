# <a name="title-storage-uris"></a>标题存储 URI
 
本部分提供有关统一资源标识符 (URI) 地址和关联的超文本传输协议 (HTTP) 方法的详细信息，从 Xbox Live*标题*存储服务。
 
所有平台上运行的游戏均可使用此服务。
 
这些 Uri 的域是 titlestorage.xboxlive.com。
