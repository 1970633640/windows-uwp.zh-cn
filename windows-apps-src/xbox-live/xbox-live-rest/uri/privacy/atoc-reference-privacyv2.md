# <a name="privacy-uris"></a>隐私 URI
 
本部分提供有关通用资源标识符 (URI) 地址和关联的超文本传输协议 (HTTP) 方法的详细信息从 Xbox Live 服务中针对*隐私*。
 
仅 Windows 8 上运行的游戏，在 Xbox 360 或 Windows Phone 设备可以使用此服务。
 
这些 Uri 的域是 privacy.xboxlive.com。
