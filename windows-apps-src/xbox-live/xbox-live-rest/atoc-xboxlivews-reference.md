# <a name="xbox-live-services-restful-reference"></a>Xbox Live 服务 RESTful 参考

Xbox Live 服务是一套用于管理游戏和玩家信息的服务。 这些服务支持**Xbox.Services** API，应尽可能使用。 使用 RESTful 接口较新的方法不会添加到**Xbox.Services**尚未此部分中所述。
