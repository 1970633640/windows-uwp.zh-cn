# <a name="xuidlist-json"></a>XuidList (JSON)
要对其执行操作的 Xuid 列表。 
<a id="ID4EN"></a>

 
## <a name="xuidlist"></a>XuidList
 
XuidList 对象具有以下规范。
 
| 成员| 类型| 描述| 
| --- | --- | --- | 
| xuid| 字符串的数组| Xbox 用户 ID (XUID) 值的应执行的操作，或应返回数据的列表。| 
  
<a id="ID4EMB"></a>

 
## <a name="sample-json-syntax"></a>JSON 语法示例
 

```json
{
    "xuids": [
        "2533274790395904", 
        "2533274792986770", 
        "2533274794866999"
    ]
}
    
```
