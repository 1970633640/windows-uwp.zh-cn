# <a name="add-and-compile-the-xbox-live-apis-source-in-your-uwp-project"></a>向 UWP 项目添加 Xbox Live API 源并进行编译

Xbox Live API (XSAPI) 源现可在 GitHub 上[ https://github.com/Microsoft/xbox-live-api ](https://github.com/Microsoft/xbox-live-api)。 开发人员可以遵循以下说明更新其项目，以使用本地生成。

如果出现以下情况，你可能要自行生成 XSAPI：
1. 如果要调试错误来了解错误代码的来源。
1. 如果我们在分发 QFE 之前，为你提供了可修复问题的源代码修补程序。

要自行编译 XSAPI 项目，请遵循 GitHub 项目站点上的说明。
